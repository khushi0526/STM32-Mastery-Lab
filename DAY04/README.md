# STM32 Day 04 – PWM LED Brightness Control

## Overview

This project demonstrates PWM (Pulse Width Modulation) in STM32 by controlling LED brightness. Instead of simple ON/OFF operation, PWM enables gradual brightness variation.

---

## Components Used

* STM32 NUCLEO C031C6
* LED
* Resistor
* Wokwi Simulator

---

## Working

The LED brightness gradually increases and decreases using PWM signals generated from the STM32 board.

This project introduced PWM control, an essential concept for motors, servos, LEDs, and embedded applications.
