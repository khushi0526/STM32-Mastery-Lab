# STM32 Day 03 – LED Toggle using Push Button

## Overview

This project demonstrates button-based LED toggling using STM32 GPIO. Unlike simple ON/OFF control, a single button press changes the LED state.

---

## Components Used

* STM32 NUCLEO C031C6
* Push Button
* LED
* Resistor
* Wokwi Simulator

---

## Working

* First press → LED ON
* Second press → LED OFF
* Repeats continuously

Basic debouncing is implemented to avoid false triggering.
